Text To LaTeX 1.0

Description: Script written in Python to convert lines of mathematical statements in English into LaTeX syntax.
Source Code: Link
Current Release:

  • Core: Allows for embedded expressions to be converted
  • Conversion Types:
    • Symbol Swapping: Simple conversion from one symbol to another.
      i.e. "infinity" -> "\infty"
    • Complex Patterns: Converts via a specific pattern.
      i.e. "integral of f(x) from 0 to 1" -> "\int\limits_0^1 f(x)"

Next Release:

  • Core: Add script to be run on this website if possible
  • Conversion Types: Add more patterns/symbols to conversion table
